位操作题目小结

关于位操作题目的个人总结。

Single Number

Given an array of integers, every element appears twice except for one. Find that single one.

同一个数xor为0，0与任何数xor为本身。

Single Number II

Given an array of integers, every element appears three times except for one. Find that single one.
这里有个小前提，就是不出现三次的数只出现一次。
可以用ones代表只出现一次的数位，twos代表出现了两次的数位，xthrees代表出现了三次的数位的取反。

Single Number III

Given an array of numbers nums, in which exactly two elements appear only once and all the other elements appear exactly twice. Find the two elements that appear only once.

For example:

Given nums = [1, 2, 1, 3, 2, 5], return [3, 5].
出现两次的xor消去，可以得到a和b的xor值。
因为a和b不等，因此必定有1位以上位数不同，我们只需找到最后1位不同即可。
用diff ＝ aXorb & -aXorb 来表示，例如3 ＝ 0011， 5 ＝ 0101， aXorb ＝ 0110.
那么 —aXrob ＝ 1010， diff ＝ 0010，也就是说，3和5最后一个不同的bit是左数第三个bit。然后将diff和数组中各个数字做与运算，结果为0 是一类，不为0是另一类。出现两次的数字会xor自然消去，因此res0和res1就分别是两个数字了！
