git使用小结

发表于 2016-07-28

什么是git

Git 是 Linus Torvalds 为了帮助管理 Linux 内核开发而开发的一个开放源码的版本控制软件。

为什么要用git

在实际开发中，经常需要对代码进行迭代，版本控制如果靠一份一份复制命名v1 v2是很让人伤神的。同时，如果有多个开发者，如何对代码进行合并也是非常重要的。因此熟练的使用git是程序员的自我修养。

git安装

Linux和MacOS下，使用包管理工具进行安装即可。Windows下去网站下载即可，用git bash像Unix的Terminal中那样使用git。

安装完成后记得设置你的身份：

如何使用git

本地使用git的话，在新建的文件夹目录中执行 git init，便会初始化该仓库。添加文件使用 git add filename。注意添加完后要执行 git commit 来完成这次对仓库的修改，此时要加 -m “”附带提交说明，提交说明用来帮助说明这次修改的原因。

使用 git status 查看仓库状态，git show 查看上次commit的具体内容。git当然还支持版本回退的功能，使用git reset即可，这里具体不做说明。

使用远程仓库

远程仓库常见的有github，gitlab等。功能大同小异，任君选择。
将用户主目录下的SSH key 添加到github的SSH keys内，就可以无需输入密码地使用.git结尾的链接来clone，push等一系列操作了。
